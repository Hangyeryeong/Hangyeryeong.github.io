### DHCP Server 방법<br/>

```yaml
# mount /dev/hdc /cdrom

# cd /cdrom/CentOS
# rpm -ivh dhcp (tap)
# rpm -ivh dhcp-devel (tap)

# cd /usr/share/doc/dhcp*
# cp dhcpd.conf.sample
# vi /etc/dhcpd.conf
```
